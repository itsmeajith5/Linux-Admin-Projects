# 👤 User Creation Script

This script automates the process of creating a new Linux user, seting a password.

## 🚀 Features

- Create a new user
- Print success message

## 📦 Requirements

- Linux system
- Root or sudo privileges
- Bash shell

## 🛠️ Usage

```bash
chmod +x create_user.sh
sudo ./create_user.sh
